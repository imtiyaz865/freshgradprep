# Security Policy 🔐

React Hyderabad team takes security seriously.

If you discover a security vulnerability, please report it responsibly.

## Reporting a Vulnerability

Please **do not open a public GitHub issue** for security vulnerabilities.

Instead, report it via email:

reacthyderabad@gmail.com (placeholder)

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

We aim to acknowledge security reports within **48 hours** and work towards resolving them as quickly as possible.

Thank you for helping keep the project secure.
